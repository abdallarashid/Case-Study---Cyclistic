# Case-Study---Cyclistic
Cyclistic - open data
Hi, this is a case study for an imaginary company, Cyclistic, the data use in this analysis is open data. The case study is part of Google Data Analytics Course.
The data was downloaded from the provided link, and then cleaned, organized, and prepared using Power Query. I then transfer the data to SQL for analysis, and finally used the results to upload to Tableau for graphical representations, from this graphs we manage to draw conclusions on whether to proceed with marketing into converting casual bike riders into permanent members.
